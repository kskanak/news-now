Difference between var, let and const ??

var is function scope, let is block scope and const is block scope also. var is mutable but let and const is immutable, means Can redeclare same variable name with var but cant redeclare same name with let and const. property value cannot change during const its fixed, can change n reset with let and var.

Difference between Arrow function and Normal Function ??

Regular function declaration requre to write function to declare, where Arrow function doesn't requre to write Arrow. In regular function, have to use return keyword to return any value. If dont return anything then the function will return undefined. If the arrow function contains one expression, can omit the curly braces, and then the expression will be implicitly returned.

Some Idea about map, forEach, filter, and find.

Array map uses for looping through array, after the completing operation the map return the value in a array. Like array map, forEach is also uses for looping through but it doesnt return any value. Applying condion in a array, after fulfilling the condition, array filter return all the value that match the applied condition, just like array filter, array find also work on condition but unlike filter it doesnt return all the value that matched the condition, it only return the first value that match the condition.
